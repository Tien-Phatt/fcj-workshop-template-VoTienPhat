### Week 10 Objectives:

* Understand how AWS records and monitors activities across an AWS account.
* Learn how to collect, store, and analyze AWS service logs.
* Explore AWS CloudTrail and Amazon Athena for auditing and security analysis.
* Gain hands-on experience building a centralized logging and monitoring solution on AWS.

### Week 10 Achievements:

* Gained a comprehensive understanding of AWS CloudTrail and its role in auditing and monitoring AWS environments.

* Learned how to:
  * Create and configure AWS CloudTrail.
  * Store activity logs in Amazon S3.
  * Monitor API calls and account events.

* Understood how Amazon Athena enables serverless SQL queries against CloudTrail logs stored in Amazon S3.

* Successfully completed hands-on exercises involving:
  * Creating AWS CloudTrail Trails.
  * Configuring log delivery to Amazon S3.
  * Creating Athena databases and tables.
  * Querying and analyzing CloudTrail logs.

* Learned how to combine AWS CloudTrail, Amazon S3, and Amazon Athena to build a centralized logging and auditing solution.

* Improved the ability to investigate AWS account activities and support security monitoring, compliance, and operational troubleshooting.

* Successfully completed the lessons and hands-on labs covering **AWS CloudTrail**, **Amazon Athena**, and **AWS Logging & Auditing** in the First Cloud Journey Bootcamp.
