### Kết quả đạt được tuần 7:

* Hiểu sự khác nhau giữa các dịch vụ lưu trữ dạng File trên AWS.

* Nắm được nguyên lý hoạt động của:
  * Amazon Elastic File System (EFS)
  * Amazon FSx
  * AWS DataSync
  * AWS Transfer Family

* Biết cách lựa chọn dịch vụ lưu trữ phù hợp theo từng nhu cầu triển khai:
  * Linux Workloads
  * Windows File Server
  * HPC (High Performance Computing)
  * Chia sẻ dữ liệu giữa nhiều máy chủ

* Thực hành thành công việc:
  * Tạo Amazon EFS.
  * Kết nối EC2 với EFS.
  * Triển khai Amazon FSx.
  * Đồng bộ dữ liệu bằng AWS DataSync.

* Hiểu quy trình truyền dữ liệu giữa hệ thống On-premises và AWS một cách an toàn và hiệu quả.

* Nắm được các phương pháp quản lý và chia sẻ dữ liệu tập trung cho nhiều máy chủ trong môi trường Cloud.

* Hoàn thành các bài học và bài thực hành về File Storage Services trong chương trình First Cloud Journey Bootcamp.
