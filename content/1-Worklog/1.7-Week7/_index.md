  * Amazon Elastic File System (Amazon EFS)
  * Amazon FSx
  * AWS DataSync
  * AWS Transfer Family

  * Linux applications
  * Windows File Server environments

  * Creating Amazon EFS file systems.
  * Mounting Amazon EFS on EC2 instances.
  * Deploying Amazon FSx.
  * Synchronizing data using AWS DataSync.
