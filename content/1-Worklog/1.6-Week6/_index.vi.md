### Kết quả đạt được tuần 6:

* Hiểu các nhóm dịch vụ lưu trữ nâng cao trên AWS và lựa chọn dịch vụ phù hợp với từng nhu cầu sử dụng.

* Nắm được sự khác biệt giữa:
  * Amazon S3 Storage Class.
  * Amazon Glacier.
  * AWS Snow Family.
  * AWS Storage Gateway.

* Hiểu quy trình xây dựng hệ thống sao lưu dữ liệu bằng AWS Backup.

* Thực hành thành công:
  * Tạo Backup Plan.
  * Cấu hình Backup Vault.
  * Thực hiện Restore dữ liệu.
  * Theo dõi trạng thái Backup.

* Hiểu quy trình di chuyển máy chủ từ môi trường On-premises lên AWS thông qua VM Import/Export.

* Thực hành triển khai Amazon EC2 từ Virtual Machine đã được chuyển đổi thành AMI.

* Nắm được quy trình Migration cơ bản, từ việc chuẩn bị máy chủ nguồn, chuyển dữ liệu lên Amazon S3 đến triển khai máy chủ trên AWS.

* Hoàn thành toàn bộ các bài học và bài thực hành về Storage Services, Backup và Migration trong chương trình First Cloud Journey Bootcamp.
