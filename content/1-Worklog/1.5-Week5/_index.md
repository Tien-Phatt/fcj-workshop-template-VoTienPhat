### Week 5 Objectives:

* Understand AWS storage services and data-sharing solutions.
* Learn how to integrate on-premises storage with AWS using AWS Storage Gateway.
* Explore static website hosting with Amazon S3 and content delivery using Amazon CloudFront.
* Gain hands-on experience deploying and managing storage services on AWS.

### Week 5 Achievements:

* Gained a solid understanding of AWS Storage Gateway and its role in integrating on-premises storage with AWS.

* Learned how to create and manage File Shares using AWS Storage Gateway.

* Understood the core concepts and architecture of Amazon S3, including buckets, objects, and storage management.

* Successfully completed hands-on exercises involving:
  * Creating Amazon S3 Buckets.
  * Uploading and managing objects.
  * Hosting a static website on Amazon S3.

* Learned how to configure Bucket Policies and Object Permissions to securely control access to stored data.

* Understood how Amazon CloudFront improves website performance by delivering content through a global Content Delivery Network (CDN).

* Successfully practiced advanced Amazon S3 features, including:
  * Bucket Versioning.
  * Cross-Region Replication (CRR).
  * Object management.

* Successfully completed the Storage Gateway, Amazon S3, and Amazon CloudFront lessons and hands-on labs in the First Cloud Journey Bootcamp.
