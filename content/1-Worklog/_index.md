## Introduction

During my internship at **First Cloud AI Journey Bootcamp** (from **April 17, 2026** to **July 30, 2026**), I participated in a structured AWS training program designed to build a solid foundation in **Cloud Computing** and **Amazon Web Services (AWS)**. The program combined theoretical knowledge with hands-on laboratory exercises, allowing me to gain practical experience in deploying, configuring, and managing cloud resources.

This worklog summarizes my learning journey over **12 weeks**. Throughout the program, I studied a wide range of AWS services, including Networking, Compute, Storage, Security, Identity and Access Management (IAM), Logging, Monitoring, and other essential cloud technologies. Each week focused on a specific topic, accompanied by hands-on labs to reinforce the concepts learned.

For every week, I defined learning objectives, completed the required lessons and practical exercises, and documented the knowledge and skills I acquired. This worklog not only records my progress throughout the internship but also demonstrates the practical experience I gained while preparing for real-world cloud projects.

The main activities completed during each week are summarized below:

**Week 1:** [Introduction to First Cloud Journey Bootcamp, Cloud Computing, and the AWS Ecosystem](1.1-week1/)

**Week 2:** [Learning Amazon VPC and the Fundamental Networking Components on AWS](1.2-week2/)

**Week 3:** [Exploring AWS Networking Solutions: VPC Peering, Transit Gateway, Hybrid Networking, and Amazon Route 53](1.3-week3/)

**Week 4:** [Learning AWS Compute Services and Deploying Amazon EC2 Instances](1.4-week4/)

**Week 5:** [Exploring AWS Storage Services: AWS Storage Gateway, Amazon S3, and Amazon CloudFront](1.5-week5/)

**Week 6:** [Learning AWS Backup, Disaster Recovery, and Migration Solutions](1.6-week6/)

**Week 7:** [Studying File Storage and Data Transfer Services: Amazon EFS, Amazon FSx, AWS DataSync, and AWS Transfer Family](1.7-week7/)

**Week 8:** [Learning AWS Security Fundamentals, IAM, and AWS Key Management Service (AWS KMS)](1.8-week8/)

**Week 9:** [Hands-on Practice with AWS Lambda, Resource Tags, and Advanced IAM Policy Management](1.9-week9/)

**Week 10:** [Learning AWS CloudTrail, Amazon Athena, and Building a Logging & Auditing Solution](1.10-week10/)

**Week 11:** [Strengthening Identity and Access Management with IAM Groups, IAM Roles, and Switch Role](1.11-week11/)

**Week 12:** [Advanced IAM Practices with Access Keys, IAM Roles, and AWS Security Best Practices](1.12-week12/)

By the end of the internship, I had developed a solid understanding of AWS core services and gained valuable hands-on experience in designing, deploying, securing, and managing cloud infrastructure. The knowledge and practical skills acquired throughout this program also provided a strong foundation for my participation in the **AI AWS Architecture Reviewer** project and my continued professional development in Cloud Computing and AWS technologies.
